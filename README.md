# Customer Behavior Data Analysis Project

## Project Overview
This project analyzes customer shopping behavior using Excel, SQL, Python, and Power BI.

## Tools Used
- Excel
- Python (Pandas)
- MySQL
- Power BI

## Files Included
- customer_shopping_behavior.csv
- customer_shopping_behavior.sql
- project.ipynb
- Customer Behavior Dashboard.pbix
- Business Problem Document.pdf
- Shopping Insights.pptx

## Dashboard Insights
- Customer purchase trends
- Category-wise sales analysis
- Gender-wise shopping behavior
- Age group analysis
- Revenue insights
